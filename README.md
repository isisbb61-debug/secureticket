# SecureTicket – SEO-Oriented Static Website

**Website:** https://dundunticket.com

SecureTicket is a **static HTML website** designed to study and implement
**Google SEO–friendly information architecture** in a YMYL-related topic.

The project focuses on **content structure, internal linking, and crawl efficiency**
rather than visual effects or dynamic features.

---

## Purpose

- Build a clean, SEO-focused static site
- Test hub–category–detail page architecture
- Optimize internal linking without keyword cannibalization
- Improve crawlability and index stability
- Maintain a neutral, information-only presentation

> This project does **not** sell or promote financial products or services.

---

## Key SEO Strategies

- **Single-topic focus** (topical authority)
- **Hub & spoke structure**
  - Main hub page (`/`)
  - Category index pages
  - Detail pages linking back to hub
- Minimal cross-linking between sibling pages
- Descriptive, human-readable URLs
- Canonical URLs on all pages
- Page-level unique `<title>` and `<meta description>`

---

## Tech Stack

- HTML5 (static pages only)
- CSS3 (single global stylesheet)
- No JavaScript framework
- No build tools
- Fully compatible with:
  - Google Search
  - GitHub Pages
  - Cloudflare Pages
  - Netlify

---

## UI / UX Principles

- Mobile-first layout
- High readability (news-style typography)
- No aggressive CTAs
- No ads or affiliate elements
- Fast load time (low LCP / CLS)
